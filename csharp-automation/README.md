# C# Playwright Automation

This folder is intended to hold the C# version of the same Sauce Demo automation implemented in `js-ts-automation`.

## Suggested workflow

1. Install .NET SDK (6.0 or later).
2. Run `dotnet new console -n CSharpPlaywright` inside this folder.
3. Add Playwright support with `dotnet add package Microsoft.Playwright`.
4. Create page objects and tests to mirror the existing JS/TS suite.
